## Introduction 
A smart robot used for surveillance and monitoring
Controlled using a smartphone via Bluetooth
Equipped with a WiFi camera for live streaming
Can operate in low-light environments
## Objectives
To build a wireless spy robot using ESP32
To enable remote control via mobile app
To integrate live video streaming
To detect obstacles automatically
##  Components
Ultrasonic Sensor (for obstacle detection)
ESP32 Microcontroller (main controller)
L298N Motor Driver Module
DC Gear Motors with Wheels
Battery (3.7V / 7.4V)
Jumper Wires
Camera (WiFi-enabled)
Power Bank
## Working Principle
ESP32 creates a Bluetooth server
Mobile app connects to ESP32
Commands are sent from phone
Motor driver controls movement
Ultrasonic sensor detects obstacles
Camera streams live video via WiFi
## Control Commands
A → Automatic Mode
M → Manual Mode
F → Forward
B → Backward
R → Rotate
S → Stop
